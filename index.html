Projetos desenvolvidos no módulo 1


O principal foco no módulo 1 é ensinar a criança os principais conceitos de programação, como sequência, algoritmo, binários, eventos e variáveis


Projetos desenvolvidos no módulo 2


O principal foco no módulo 2 é reforçar os conceitos aprendidos no módulo 1, e ensinar sobre condicionais if, loop for, loop while, conectivos "e" e "ou".


Projetos desenvolvidos no módulo 3

O principal foco no módulo 3 é ensinar os alunos a fazer o básico de desenvolvimento web e desenvolver jogos em HTML5. Nesse percurso vai desenvolver habilidades de HTML (e suas tags), vai estilizar o site utilizando CSS (com seus seletores) e Javascript. Além disso, utiliza Bootstrap para criação de sites responsivos e Machine Learning (teacheablemachine, mobileNet, poseNet, CocoSSD) para aprender sobre redes neurais, banco de dados e IA.
